# Code

This directory contains the source code and related components for the Sovereign AGSI system.

- **/hyperon-bridge**: Contains the code for the Hyperon Bridge, a component responsible for connecting different parts of the AGSI system.
- **/phoenix-nexus**: Contains the code for the Phoenix Nexus, the central hub for communication and data exchange between Phoenix Nodes.
- **Universal_API_Schema.json**: A JSON file defining the universal API schema used across the AGSI system for interoperability.

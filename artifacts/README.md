# Artifacts

This directory contains immutable, cryptographically-verified artifacts that form the evidential backbone of the Sovereign AGSI project. These files provide proof of concept, authorship, and existence.

- **/ledgers**: Contains cryptographic ledgers and signature files, providing a verifiable chain of custody for all system events and declarations.
- **/pdfs**: Contains key architectural documents, manifestos, and whitepapers in PDF format.
- **/timestamps**: Contains OpenTimestamps (.ots) files, which anchor the project's documents to the Bitcoin blockchain for immutable proof of existence.
